Opdracht 3

Er zijn aldus 14 verschillende pseudo elementen. Je hebt ze in de voorgaande opdracht reeds opgezocht en uitgezocht wat ze doen.
Kun je van al deze pseudo-elementen een voorbeeld maken?

::first-line

Het pseudo-element ::first-line zorgt ervoor dat je de eerste regel in je content onafhankelijk van de rest van je styling kunt aanpassen.
Een voorbeeld daarvan kun je al in deze alinea zien.
Als alles goed is gegaan zal de eerste regel van deze alinea een blauwe achtergrond hebben en witte letters.
Deze aanpassing is gemaakt met deze CSS values:

.firstline > p::first-line{
background-color: dodgerblue;
color: white;
}

Dit zijn de mogelijke properties voor het ::first-line element:


font properties
color properties
background properties
word-spacing
letter-spacing
text-decoration
vertical-align
text-transform
line-height
clear



::first-letter

Het first letter element zorgt ervoor dat de eerste letter in de aangewezen selectors een individuele opmaak ontvangt.

::before en ::after

Deze twee elementen voeg ik even samen voor het gemak, aangezien beide elementen aardig veel op elkaar lijken.

Met deze elementen kun je namelijk een element toevoegen voor en na een html-element.

Zo is er een afbeelding voor en na de titel toegevoegd van deze alinea.

::selection

Het selection element kan de opmaak aanpassen wanneer je een stuk tekst/content selecteert.
Als je probeert een stuk tekst in deze alinea te selecteren, zul je zien dat de kleur van het geselecteerde stuk tekst groen gemarkeerd wordt en de tekst wit wordt.
